# ListeApp

Application Android simple pour gérer une liste de tâches.

## Fonctionnalités

- Affichage d'une liste de tâches
- Ajout de nouvelles tâches avec nom et description

## Installation

1. Clonez ce dépôt
2. Ouvrez le projet dans Android Studio
3. Synchronisez le projet avec les fichiers Gradle
4. Compilez et exécutez l'application

## Utilisation

1. Au lancement, l'application affiche une liste de tâches
2. Pour ajouter une tâche, cliquez sur le bouton "Ajouter"
3. Remplissez le nom et la description, puis validez

## Technologies

- Kotlin
- XML Layouts
- Gradle
- API Minimum: Android 7.0 (API 24)
